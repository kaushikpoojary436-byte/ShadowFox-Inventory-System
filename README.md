# Inventory Management System

## Description
This is a simple Inventory Management System developed using Java Swing GUI.

## Features
- Add Items
- Delete Items
- GUI Interface using Swing
- Inventory Table

## Technologies Used
- Java
- Java Swing
- VS Code

## How to Run

Compile:
```bash
javac InventoryGUI.java
```

Run:
```bash
java InventoryGUI
```
